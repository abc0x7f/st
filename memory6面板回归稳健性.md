## 已确认的方法口径

### 1. 缩尾处理

- 缩尾处理不是删除某个“省份-年份”整条样本。
- 标准做法是对连续变量的极端值进行分位点替换。
- 可处理变量包括：
  - `eff`
  - `lntl`
  - `ind`
  - `urb`
  - `rd`
  - `open`
  - `es`
- 对当前 `240` 个样本而言：
  - `1%` 双侧缩尾改动较少，属于温和稳健性检验；
  - `5%` 双侧缩尾力度更强，适合作为补充检验。
- 当前建议口径：
  - 正文可报告 `1%` 缩尾；
  - 同时补充 `5%` 缩尾结果，以增强说服力。

### 2. `lntl` 滞后一期

- 滞后一期的主要作用：
  1. 缓解反向因果；
  2. 弱化同期联立偏误；
  3. 更符合夜间灯光聚合度对碳排放效率影响存在传导时滞的经济逻辑；
  4. 可作为稳健性检验。
- 滞后模型形式：

$$
eff_{it}=\alpha+\beta_1 lntl_{i,t-1}+\gamma'X_{it}+\mu_i+\delta_t+\varepsilon_{it}
$$

- 若样本期为 `2015—2022`，则构造 `lntl_{i,t-1}` 需要 `2014` 年灯光数据。
- 当前已经具备 `2014—2022` 的灯光数据，因此可以为 `2015—2022` 样本生成完整的 `lntl_lag1`。

### 3. 替换被解释变量口径

- 可以将原有 `SBM` 口径替换为省级 `NDDF` 超效率 `DEA` 口径，作为稳健性检验。
- 关键前提：
  - 仍然使用省级样本；
  - 不建议直接用地级市 `NDDF` 碳排放效率替换当前省级被解释变量。
- 原因：
  - 当前主模型是省级面板；
  - 若改为地级市口径，将改变研究层级，不再属于单纯的稳健性检验。
- 因此，待做方案应为：
  - 重新以省级数据测算 `NDDF` 碳排放效率；
  - 用新 `eff` 替换原 `eff` 回归。

### 4. 异质性检验

- 已纳入后续分析计划。
- 可作为主结论的扩展分析部分。
- 当前对话尚未具体锁定分组方案，后续需根据研究逻辑与数据条件细化。

## 本轮对话新增结论与口径

### 1. 基准回归的推断口径已改为 `Driscoll-Kraay`

- 原先基准回归脚本使用的是：
  - `statsmodels OLS`
  - `C(province) + C(year)` 显式虚拟变量
  - 默认普通 OLS 标准误
- 现在统一改为：
  - `PanelOLS`
  - 双向固定效应
  - `cov_type="driscoll-kraay"`
- 这意味着：
  - **模型设定并没有变**，仍然是双向固定效应；
  - **主要变化在标准误口径**，不是因为 `上海-2020` 那一条 `lntl` 微调导致显著性大变。

### 2. 旧基准结果与新基准结果的差别来源

- 旧基准回归中：
  - `lntl` 系数约为 `0.172489`
  - `p ≈ 0.0072`
- 新口径下：
  - `lntl` 系数约为 `0.172467`
  - `p ≈ 0.0504`
- 判断：
  - 系数几乎没变；
  - 差异主要来自标准误由普通 OLS 改成了 `Driscoll-Kraay`；
  - 因此不能把两个 `p` 值当成同一统计口径直接比较。

### 3. 当前基准回归的主要结果

- 当前统一口径下，基准模型核心变量结果为：
  - `lntl = 0.172467`
  - `std_err = 0.087610`
  - `t = 1.9686`
  - `p = 0.050404`
- 控制变量结果大致为：
  - `urb` 显著为负；
  - `open` 显著为正；
  - `rd` 为正，边际显著；
  - `ind`、`es` 不显著。

### 4. 当前稳健性检验的主要结果

- `baseline`
  - `lntl = 0.172467`
  - `p = 0.050404`
- `1%缩尾`
  - `lntl = 0.175806`
  - `p = 0.034680`
- `5%缩尾`
  - `lntl = 0.170266`
  - `p = 0.049389`
- `滞后一期`
  - `lntl_lag1 = 0.221492`
  - `p = 0.004028`

结论：

- 缩尾前后核心系数方向一致且量级接近；
- 滞后一期后仍显著为正，且显著性增强；
- 说明主结论并非由少数极端值驱动，且在时间上具有延续性。

### 5. 当前对残差图、Q-Q 图与 `R^2` 的解释口径

- Q-Q 图出现更明显的 `S` 型，不直接说明固定效应没有捕捉好；
- 更合理的解释是：
  - 残差存在厚尾、偏态或少数异常点；
  - 当前模型已识别出异方差、序列相关及潜在截面相关，因此残差图不必被过度解读为模型无效。
- 当前 `PanelOLS` 下的 `R^2` 与原来 `OLS + 虚拟变量` 的 `R^2` 不可直接比较：
  - 原先高 `R^2` 大量来自固定效应本身；
  - 现在的 `R^2` 更偏向反映控制固定效应后解释变量对有效变动的解释程度。

### 14.5 “全国总体显著，但分区域后多数不显著”是否合理

该结果是合理的，并不与总体回归结论矛盾，原因主要有以下几点：

1. **分组后样本量明显下降，统计功效减弱**

全国样本量为 `240`，分组后分别变为东部 `80`、中部 `48`、西部 `88`、东北 `24`。样本拆分后，标准误通常会增大，导致 `t` 值下降，因此即便系数仍为正，也可能不再显著。

2. **全国回归反映的是平均意义上的总体效应**

总体模型显著，说明在全国层面上，`lntl` 对 `eff` 存在平均正向影响。只要各区域系数方向大致一致，合并样本后就可能因估计更稳定而表现为显著。当前四个区域系数均为正，因此总体显著具有可解释性。

3. **区域内部依然存在较强异质性**

即使已经按东中西东北分组，每个大区内部仍包含发展阶段、资源禀赋、产业结构差异很大的省份。组内差异会抬高估计噪声，从而削弱单组回归显著性。

4. **总体结果本身属于边际显著**

基准回归中，全国总体 `lntl` 的估计值为 `0.1725`，`p = 0.0504`，本身就接近 5% 临界值。在这种情况下，分组后显著性下降是很常见的。

因此，更稳妥的解释不是“东中西没有作用，只有东北有作用”，而应写为：

> 全国层面上，夜间灯光对绿色经济效率具有显著正向影响；分区域回归后，东部、中部和西部地区的核心解释变量系数虽仍为正，但统计证据不足，东北地区则表现出更强且显著的促进作用。这说明夜间灯光对绿色经济效率的影响具有明显区域异质性，同时分组后样本量下降和组内差异扩大也削弱了参数估计精度。

### 14.6 论文写作时的注意点

1. 不宜将结果表述为“东中西无效、东北有效”，而应强调“方向一致但显著性和作用强度存在差异”。
2. 东北仅包含 `3` 个省份，样本量最小，因此即便结果显著，也应避免过度外推。
3. 组间“一个显著、一个不显著”并不自动等于“两个区域之间差异显著”，若要更严格比较区域差异，后续可考虑加入区域交互项进行正式检验。

### 14.7 后续可选扩展

若需要进一步增强异质性分析的统计说服力，可在双固定效应模型中加入区域交互项：

$$
eff_{it}=\alpha+\beta lntl_{it}+\sum_r \theta_r (lntl_{it}\times Region_r)+\gamma'X_{it}+\mu_i+\delta_t+\varepsilon_{it}
$$

该方法可以直接检验不同区域之间 `lntl` 系数差异是否显著，比简单比较分组回归结果更规范。
