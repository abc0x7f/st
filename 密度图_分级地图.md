# 图表绘制规范

---

## 一、变量分布密度图（直方图 + 核密度曲线）

### 适用场景

对一组或多组连续型变量绘制直方图叠加核密度曲线，用于检验分布形态（偏态、峰度、多峰性）。

### 绘图规范

#### 1. 布局

- 多变量并排放置，`1 × N` 子图排列（N 为变量数）。
- 每个子图采用**双轴**结构：左轴 = 频数（直方图），右轴 = 密度（核密度曲线）。
- 整体加 `suptitle` 标注图组主题。

#### 2. 直方图

- 分箱数量：默认 30 个等宽 bin。
- 柱体颜色：使用 matplotlib 渐变色板，按柱体从左到右渐变。
- 柱体边框：白色细线（`edgecolor="white", linewidth=0.6`），增强分隔感。
- 若变量含义上不应出现负值（如原始量级指标），则裁剪 `x < 0` 的部分，设 `xlim(left=0)`。
- 若变量值全为正，也建议将横轴起点设为 `0`，避免出现负数区域干扰阅读。

#### 3. 核密度曲线（KDE）

- 绘制在右轴（`twinx()`）上，线宽 ≥ 2，颜色比直方图深一档。
- **clip 范围**：从 `bin_edges[1]`（第二个柱的左边缘）到 `values.max()`，确保曲线从直方图主体区域起画，不在左端因边界效应产生人为上升段。
  - 对右偏分布（如原始 ntl），此处理使 KDE 呈现单调递减的期望形态。
  - 对近似正态分布（如 lntl），此处理避免左端尾部过早拉低曲线。
- 右轴**关闭网格线**（`ax_density.grid(False)`），避免灰色横线横穿图面。
- 右轴底部 spine 隐藏（`spines["bottom"].set_visible(False)`），x 轴只保留左轴的。

#### 4. 图例

- 在每个子图右上角合并左轴（频数柱）与右轴（核密度曲线）的图例。

#### 5. 参考配色方案

自由选择

#### 6. 示例调用伪码

```python
draw_gradient_histogram(
    ax, values, title, xlabel,
    cmap_name="COLOR",       # 渐变色板
    kde_color="#XXXXXX",     # KDE 曲线色
    clip_negative=True,      # 是否裁剪 x<0
    force_xlim_left=0,       # 强制横轴起点
)
```

## 二、省级分级填色地图（Choropleth Map）

### 适用场景

将某一省级面板变量按等距或自定义分段着色，生成符合学术规范的中国省级 choropleth 地图。

### 绘图规范

#### 1. 底图要求

- 使用 GeoJSON 格式的中国省级行政边界文件（WGS84 坐标）。
- 需包含"南海诸岛"（九段线）要素，通常以 `adchar="JD"` 或空名称标识。
- 若绘制中国标准省级地图，建议底图中可识别出：
  - 广东
  - 广西
  - 海南
  - 南海诸岛（九段线）

#### 1.1 字体要求

- 全图文字默认使用：
  - 西文与数字：`Times New Roman`
  - 中文：`SimSun`
- 若本机缺失，可临时回退到兼容字体，但正式成图建议保持上述组合。

#### 2. 投影与比例

- 使用经纬度直投（无投影），通过 `set_aspect(1 / cos(center_lat))` 校正纬度形变，避免地图被拉伸变窄。
  - `center_lat` 取地图可视范围纬度中值（中国一般取 ~35°N）。
- 默认显示范围：经度 73°–136°E，纬度 15°–55°N。

#### 3. 省份填色

- 配色方案：使用连续色板（如 `YlOrRd`），搭配 `BoundaryNorm` 做分段着色。
- 分段规则：根据变量实际范围等距分为 5–7 段，段界取整数或 0.5 步长的美观刻度。
- 无数据省份统一填灰色（`#D9D9D9`）。
- 若变量主题偏“绿色效率”“生态效率”“碳排放效率”，优先使用**绿色连续色板**（如 `Greens`）。

#### 4. 省份边界

- 所有省份边界使用**黑色加粗边框**（`edgecolor="black", linewidths=1.2`）。

#### 5. 经纬度边框与刻度

- 图面四周保留黑色粗边框（spine `linewidth=2.5`）。
- 四边均显示**主刻度**（每 5° 一个，朝内，长 6px）和**小刻度**（每 1° 一个，朝内，长 3px）。
- 主刻度标签格式：经度 `XXX°E`，纬度 `XX°N`。
- 关闭背景网格（`ax.grid(False)`）。

#### 6. 南海诸岛小图（标准中国地图格式）

- 在地图**右下角**以 inset axes 形式绘制南海诸岛缩略图。
- 内容建议包括：
  - 广东南部
  - 广西南部
  - 海南省
  - 九段线岛礁
- 不建议只画海南和九段线，否则海南北侧参照海岸会缺失，视觉上容易显得突兀。
- 当前验证效果较好的显示范围为：
  - 经度 `104°–123°E`
  - 纬度 `2°–23°N`
- 若只想保留广东、广西下半部，可进一步下压上边界，但应保证海南与大陆南缘关系仍可辨认。
- 黑色边框包围（`linewidth=1.8`），标题"南海诸岛"。
- 位置不得与主图外框重叠。
- 当前验证较好的 inset 布局参数为：
  - `SCS inset = [0.71, 0.14, 0.14, 0.22]`

#### 7. 指北针与图例区

##### ① 指北针
- 黑色向上箭头 + 顶部"N"字样。
- 放置在**主地图大边框内左上角**，与边框保持适当留白，不压住省界或标题。

##### ② 图例区（左下角）中的比例尺
- 双段交替黑白矩形条。
- 长度需**计算得出**：根据比例尺所在纬度 `lat_ref` 计算 `1° lon = 111.32 × cos(lat_ref) km`，反推目标公里数对应的经度度数，再换算到图面单位。
- 标注 `0`、中点公里数、总公里数（如 `0 / 250 / 500 km`）。
- 数字标注与矩形条之间留足间距，不得重叠。

##### ③ 图例区（左下角）中的分段色块图例
- 每段一行：左侧色块矩形 + 右侧范围文字（如 `0.0 – 0.5`）。
- 最下方添加**变量名称说明**文字（如"夜间灯光聚合度对数"），粗体居中。

#### 8. 布局防重叠要点

| 元素 | 注意事项 |
|------|---------|
| 南海诸岛 inset | 距主框边缘留 ≥ 3% figure 宽度的间距 |
| 指北针 | 位于主图左上角框内，不与标题、省界和经纬度刻度重叠 |
| 比例尺文字 | 数字标注下移至矩形条下方 0.4–0.5 个单位 |
| 分段色块 | 各行间距均匀（box_h + 0.25），最底行下方留文字空间 |

#### 9. 参考参数一览

```
figure size    = 14 × 11 inches
main axes      = [0.09, 0.10, 0.78, 0.82]   (left, bottom, width, height)
SCS inset      = [0.71, 0.14, 0.14, 0.22]
SCS xlim       = [104, 123]
SCS ylim       = [2, 23]
legend axes    = [0.09, 0.10, 0.18, 0.36]
compass pos    = 主图大边框内左上角
spine width    = 2.5
province edge  = black, 1.2
SCS edge       = black, 0.6
major tick     = in, 6px, 1.2w, 四边
minor tick     = in, 3px, 0.8w, 四边
font family    = Times New Roman + SimSun
color map      = Greens（效率主题优先）/ YlOrRd（一般主题） + BoundaryNorm
```

## 三、使用方式

将本文档与待绘制的数据一起提供给 AI 助手，说明：

1. **数据格式**：包含 `province`、`year`、`目标变量` 的 CSV 面板数据。
2. **GeoJSON 底图路径**。
3. **指定绘制图表类型**：密度图 / 分级地图 / 两者皆需。
4. **可选定制**：变量名称、配色方案、分段区间、比例尺长度、南海诸岛 inset 范围、字体方案等。

### 推荐提示词补充

若要快速复刻当前风格，建议在提示词里明确写出：

1. 主图字体使用 `Times New Roman + SimSun`
2. 若为效率类变量，主色调使用 `Greens`
3. 南海诸岛 inset 保留广东南部、广西南部、海南和九段线
4. inset 参数优先参考：
   - `position = [0.71, 0.14, 0.14, 0.22]`
   - `xlim = [104, 123]`
   - `ylim = [2, 23]`
5. 指北针放主图左上角框内
6. 左下角保留比例尺和分段图例

即可获得风格一致的学术级图表。
