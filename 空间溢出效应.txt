　　1. 空间相关性检验：
先做全局Moran’s I ，优先分别检验这俩：eff，lntl

1. 先基于空间权重矩阵计算每一年 eff 的全局 Moran’s I
2. 输出每年的Moran’s I、z值、p值
3. 画Moran’s I年度变化表或汇总表
4. 如果可以再画Moran散点图，至少选 2015、2018、2022 三个年份展示

判断标准：
Moran’s I显著大于0，说明存在正向空间相关
如果是大多数年份显著，说明做空间模型是有依据的

　　5. 构建空间权重矩阵

1. 构建30×30省际邻接矩阵W
2. 有共同边界记为1，否则记为0
3. 对角线全部设为0
4. 最后做行标准化处理，得到标准化矩阵W
5. 保存原始邻接矩阵和标准化矩阵两个版本

注意：
·海南与其他省份无陆地接壤，矩阵里一般全 0，需要检查空间软件是否允许孤立点
·如果孤立点影响模型运行，可以考虑在稳健性里补一个地理距离矩阵，但正文主模型仍先用邻接矩阵
·省份的顺序要按照我们之前province的表顺序一样，要不然的话矩阵数据会错位

　　6. 空间面板模型设定
在确认存在空间相关性后，优先跑空间杜宾模型SDM
模型形式建议设为双固定效应：省份固定效应 + 年份固定效应

1. 先跑普通面板回归作为对照
2. 再跑SAR、SEM、SDM三种模型
3. 以SDM为主模型
4. 固定效应优先选双固定效应
5. 若软件支持，做 LM、Robust LM、LR 或 Wald检验，判断SDM是否可以退化为SAR或SEM
6. 如果不能退化，就保留 SDM 作为最终模型

基准空间模型：
eff_it = ρWeff_it + β1lntl_it + β2ind_it + β3urb_it + β4rd_it + β5open_it + θ1Wlntl_it + θ2Wind_it + θ3Wurb_it + θ4Wrd_it + θ5Wopen_it + μ_i + λ_t + ε_it

1. ρ：被解释变量的空间滞后系数
2. W*x：解释变量的空间滞后项
3. μ_i：省份固定效应
4. λ_t：年份固定效应

　　2. 结果解释口径

1. 解释顺序统一为：
1. 先看 Moran’s I：说明是否存在空间集聚
2. 再看 SDM 中 ρ 是否显著：说明碳排放效率是否存在空间依赖
3. 再看 lntl 的直接效应：说明夜间灯光对本省碳排放效率的影响
4. 再看 lntl 的间接效应：说明夜间灯光对周边省份碳排放效率是否存在溢出
5. 最后看总效应：说明总体影响强度

1. 解释原则：
1. 直接效应显著为正：本省夜间灯光提升有助于提升本省碳排放效率
2. 间接效应显著为正：夜间灯光存在正向空间溢出，能带动邻近省份效率改善
3. 间接效应显著为负：可能存在虹吸效应、要素挤出或产业转移压力
4. 若不显著，也要如实写明，不要强行解释
