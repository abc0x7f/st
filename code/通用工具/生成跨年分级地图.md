# 跨年分级地图生成说明

## 目的

这份文档用于复刻当前项目中的“跨年省级分级地图”类图片。  
下次你只要带着**新数据**和**简短提示词**来，我就可以按接近当前项目的风格继续生成相似图片。

适用对象包括：

1. 跨年效率分级地图
2. 跨年夜间灯光分级地图
3. 跨年指数、强度、水平、得分等省级 choropleth 地图
4. 需要把多个年份拼成一张组合图的地图

---

## 一、你下次需要提供什么

### 1. 最少需要的数据字段

输入数据至少应包含以下字段：

- `province`：省份名称
- `year`：年份
- `value`：需要映射着色的指标值

如果你的变量名不是 `value`，也没关系，只要明确告诉我真实字段名即可。

### 2. 省份名称格式要求

建议使用与当前项目一致的简称口径，例如：

- 北京
- 天津
- 上海
- 重庆
- 内蒙古
- 广西
- 宁夏
- 新疆

如果你给的是全称，例如 `北京市`、`宁夏回族自治区`，我也可以做映射，但最好提前说明。

### 3. 建议同时告诉我的信息

为了少返工，最好一起告诉我：

1. 想画哪些年份  
   例如：`2015、2018、2022`
2. 指标名是什么  
   例如：`碳排放效率`、`夜间灯光对数`
3. 是否要统一分级区间  
   通常跨年比较时建议统一
4. 输出位置  
   例如：`outputs/空间分析/XX_跨年分级地图`
5. 是否要组合成一张图  
   通常推荐做成“厂字型”或 `2 x 2` 留白布局

## 二、默认复用的成图风格

如果你没有额外说明，我默认沿用当前项目的以下风格：

### 1. 字体

- 英文与数字：`Times New Roman`
- 中文：`SimSun`

### 2. 地图底图

- 使用中国省级 GeoJSON
- 主图保留全国省界
- 每个子图左下角保留“南海诸岛” inset

### 3. 组合布局

若为 3 个年份，默认使用：

1. 左上：第 1 年
2. 右上：第 2 年
3. 左下：第 3 年
4. 右下：图例、比例尺、脚注

也就是常说的“厂字型”布局。

### 4. 图例区

- 图例放在右下空白区
- 图例标题居中
- 比例尺放在图例上方
- 脚注放在图例下方

### 5. 比例尺处理

比例尺不手工乱放，按地图实际宽度计算。

核心思路是：

$$
1^\circ \text{经度长度} = 111.32 \times \cos(\text{lat\_ref}) \text{ km}
$$

再根据主图显示经度范围，把目标公里数换算成图面长度。

这意味着：

1. `500 km`
2. `1000 km`
3. `2000 km`

这些比例尺长度都应和主地图实际尺度一致，而不是视觉上“差不多”。

## 三、跨年分级地图的推荐规则

### 1. 分级区间

跨年图默认建议**所有年份共用一套分级区间**，这样才能比较。

常见做法：

1. 根据全样本最小值和最大值等距分 5 级
2. 根据业务含义手工指定分段
3. 使用固定阈值，例如：
   - `0.2`
   - `0.4`
   - `0.6`
   - `0.8`

如果你没指定，我通常会先检查变量分布，再决定是否用：

- 等距分级
- 自定义美观分级

### 2. 无数据处理

- 无数据省份统一浅灰色
- 台湾、西藏等若不在样本中，也按缺失色处理

### 3. 配色建议

如果你没指定，通常按变量主题选色：

- 效率、绿色、生态类：`Greens`
- 热度、强度、压力类：`YlOrRd`
- 发散型高低对比：`RdYlBu` 或 `RdBu`

### 4. 标题格式

通常写成：

- `2015 年`
- `2018 年`
- `2022 年`

组合图总标题写成：

- `图XX XXX 的跨年分级地图（2015、2018、2022）`
